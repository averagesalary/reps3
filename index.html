Note: This test attempts to fetch Google's homepage. A successful connection returns status 200, while a failed connection returns 404.

``` How It Works 1. The page displays a clean, modern interface with a gradient background 2. When you click the "Check Connection to Google" button, it sends a HEAD request to Google's homepage 3. If the connection is successful, it displays a green "200" status code 4. If the connection fails, it displays a red "404" status code 5. A loading animation appears while the connection is being tested The code uses the Fetch API with 'no-cors' mode to work around browser security restrictions. Note that due to CORS policies, we can't always get the exact HTTP status code from external domains, but this implementation provides a reliable way to check connectivity. You can save this code to an index.html file and open it in a web browser to test your connection to Google.
